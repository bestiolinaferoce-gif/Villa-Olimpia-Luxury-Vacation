Come inserire le variabili in n8n – Guida passo passo

Metodo A – Tramite interfaccia n8n (consigliato)
  1. Apri http://72.60.20.32:62925 nel browser
  2. Clicca su Settings (ingranaggio in basso a sinistra)
  3. Seleziona Variables
  4. Clicca Add Variable per ogni variabile
  5. Inserisci Nome e Valore come indicato nelle sezioni sopra
  6. Salva. Il workflow userà automaticamente i nuovi valori.
Metodo B – Via SSH sul server (alternativa)
  1. Connettiti al server: ssh root@72.60.20.32
  2. Modifica il file .env di n8n: nano /root/n8n/.env
  3. Aggiungi le righe:
    OPENAI_API_KEY=sk-...
    WA_TOKEN=EAA...
    RESEND_API_KEY=re_...
  4. Salva con Ctrl+X → Y → Enter
  5. Riavvia n8n: docker restart n8n
