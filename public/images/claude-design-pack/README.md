# Claude Design Pack

Curated image pack for the Villa Olimpia homepage redesign.

Use these folders as primary visual context:

- `exteriors`: property facade, entrance, overall exterior context
- `pool`: shared pool, gazebo, outdoor mood
- `interiors`: strongest interior spaces and atmosphere
- `lodges`: distinctive lodge-level details, especially Geranio, Azalea, Orchidea
- `territory`: nearby sea, Spiaggia dei Gigli, Le Castella

Design intent:

- prioritize authentic, real-property photography
- avoid generic hospitality stock aesthetics
- emphasize proximity to the sea, Mediterranean light, calm premium atmosphere
- support a homepage that feels memorable, elegant, and direct-booking oriented
