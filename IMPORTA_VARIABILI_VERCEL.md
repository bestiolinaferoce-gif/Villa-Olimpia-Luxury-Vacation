### STEP 1: Apri il File
Ho creato il file `vercel.env` nella cartella del progetto.

### STEP 2: Modifica Google Maps Key
1. Apri il file `vercel.env` 
2. Trova la riga: `NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=INSERISCI_QUI_IL_TUO_GOOGLE_MAPS_API_KEY`
3. Sostituisci `INSERISCI_QUI_IL_TUO_GOOGLE_MAPS_API_KEY` con il tuo Google Maps API Key reale

### STEP 3: Importa su Vercel
1. Vai su Vercel ‚Üí "Settings" ‚Üí "Environment Variables"
2. Clicca sul pulsante **"Import .env"** (lo vedi nella pagina)
3. Copia e incolla tutto il contenuto del file `vercel.env`
4. Clicca "Save"
5. Clicca "Redeploy" quando appare il popup
