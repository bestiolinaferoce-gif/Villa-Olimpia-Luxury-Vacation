## 📝 STEP 2: CREA TEMPLATE EMAIL

### 2.1 Vai su Email Templates
1. Nel menu a **sinistra**, clicca su **"Email Templates"**
2. Clicca sul pulsante **"Create New Template"** (in alto a destra)

### 2.2 Configura Template
1. **Nome Template:** `Prenotazione Villa Olimpia`
2. **Subject (Oggetto):** 
   ```
   Nuova Richiesta Prenotazione - {{from_name}}
   ```

3. **Content (Corpo email):** Copia e incolla questo:


## 📝 STEP 4: CONFIGURA .env.local

1. Nella root del progetto, crea/modifica il file **`.env.local`**
2. Incolla questo (sostituisci con i TUOI valori):

```env
NEXT_PUBLIC_EMAILJS_SERVICE_ID=service_bbp2k8u
NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=template_xxxxx
NEXT_PUBLIC_EMAILJS_PUBLIC_KEY=la_tua_public_key
```

**Sostituisci:**
- `service_bbp2k8u` → il tuo Service ID (quello che hai appena creato)
- `template_xxxxx` → il tuo Template ID (quello che creerai)
- `la_tua_public_key` → la Public Key che copierai
